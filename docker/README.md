# C-ISAM Docker 가이드

## 🚀 사용 방법

### 1. 이미지 빌드
```bash
./docker/build.sh
```

### 2. 데모 자동 실행
```bash
./docker/run.sh
```
- 모든 데모 프로그램이 순차적으로 실행됩니다.
- 데이터는 `docker/data` 폴더에 저장됩니다.
- 시나리오는 `docker/auto_demo.sh`를 수정하여 변경할 수 있습니다.

---

## 📦 파일 구성

| 파일 | 설명 |
|------|------|
| `build.sh` | 이미지 빌드 스크립트 (단순화됨) |
| `run.sh` | 컨테이너 실행 스크립트 (데이터 마운트 포함) |
| `auto_demo.sh` | **데모 시나리오 스크립트**. 실행 순서와 입력값 정의 |
| `data/` | 실행 결과 데이터가 저장되는 공유 폴더 |
| `Dockerfile` | Rocky Linux 8 기반 이미지 정의 |

---

## 💡 팁
- **시나리오 수정**: `docker/auto_demo.sh` 파일을 직접 수정하면, 이미지를 다시 빌드하지 않아도 변경된 시나리오가 적용됩니다.
- **데이터 확인**: 실행 후 `docker/data` 폴더에서 생성된 `.dat`, `.idx` 파일을 확인할 수 있습니다.
